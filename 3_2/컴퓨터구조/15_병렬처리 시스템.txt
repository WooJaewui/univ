
 병렬처리시스템.



 # 병렬처리의 개요.

병럴처리의 필요성.
컴퓨터시스템의 성능향상을 위한 방안.

병럴처리 parallel processing.
하나 이상의 연산을 동시에 수행함으로써, 연산속도를 증가시키려는 처리방법.

순차처리기 sequential processor.
한 번에 하나의 CPU 명령을 수행하는 기존의 컴퓨터.

병럴처리기 parallel processor.
하나 이상의 CPU 명령을 병렬로 처리하도록 설계한 컴퓨터.

성능 척도
MIPS Million Instructions Per Second : 특정기계의 명령어의 실행속도를 표시.
MFLOPS Million Floating-point Operations Per Second : 프로그램 상에서 부동소수점 연산에 기초를 둔 것.
벤치마크 benchmark : 실제 작업부하에 대한 성능을 잘 반영할 것으로 생각되는 프로그램들로 구성된 작업부하.



 # 병럴처리시스템의 분류.

병렬처리기의 분류.
동시에 처리할 수 있는 명령어나 데이터의 수, 처리기의 내부조직, 연결구조 등에 따라 분류.

분류 방법.
플린의 분류.
펭의 분류.
구조에 의한 분류.



 # 플린의 분류.

플린( Flynn )의 분류 방법.
명령어 스트림과 데이터 스트림이 컴퓨터 내에서 하나 뿐인가 혹은 여러 개 인가를 기준으로 분류.

스트림( stream )
하나의 처리기에 의하여 수행되거나 처리되는 연속적인 일련의 명령어들이나 데이터들을 의미.
단일 명령어 스트림 : 하나의 처리기 내에서 처리되는 일련의 연속적인 명령어들을 의미.
단일 데이터 스트림 : 하나의 명령어 스트림에서 요구되는 일련의 연속적인 데이터나 중간 결과를 의미.

플린의 분류에 의한 컴퓨터구조.
단일 명령어 스트림 - 단일 데이터 스트림 컴퓨터 ( SISD Single Instruction stream - Sigle Data stream )
단일 명령어 스트림 - 복수 데이터 스트림 컴퓨터 ( SIMD Single Instruction stream - Multiple Data stream )
복수 명령어 스트림 - 단일 데이터 스트림 컴퓨터 ( MISD Multiple Instruction stream - Single Data stream )
복수 명령어 스트림 - 복수 데이터 스트림 컴퓨터 ( MIMD Multiple Instruction stream - Multiple Data stream )

SISD 컴퓨터 구조. 
현재의 컴퓨터 구조.
명령어가 순차적으로 하나씩 수행된다. 
제어장치 1개, 처리기 1개.

SIMD 컴퓨터 구조.
배열처리기 구조라고도 한다.
여러 개의 처리기가 하나의 제어처리기에 의해 제어되는 구조. 
제어장치 1개, 처리기 여러개.

MISD 컴퓨터 구조.
이론상의 병렬처리구조.
전체적으로 하나의 데이터 스트림을 가지고 있는 형태.
실제로는 존재하지 않음.

MIMD 컴퓨터 구조.
다중처리기 컴퓨터 시스템이나 다중컴퓨터 시스템에 해당.
여러 개의 처리기는 각각 다른 데이터 스트림에 대하여 다른 명령어 스트림을 수행한다.



 # 펭 분류.

펭( Feng )의 분류 방법.
컴퓨터 구조를 병렬수행 정도에 따라 분류.

펭의 분류에 따른 컴퓨터의 종류.
단어별 순차, 비트별 순차처리 ( WSBS word-serial, bit-serial ) : 한 번에 한 비트씩 처리. ( 초기 컴퓨터 )
단어별 병렬, 비트별 순차처리 ( WPBS word-parallel, bit-serial ) : 여러 워드에서 한 번에 한 비트씩 병렬 처리.
단어별 순차, 비트별 병렬처리 ( WSBP word-serial, bit-parallel ) : 한 단어의 모든 비트가 병렬로 처리. 
단어별 병렬, 비트별 별렬처리 ( WPBP word-parallel, bit-parallel ) : 병렬 처리의 가능성을 최대로 높인 처리.



 # 구조에 의한 분류.

구조에 의한 분류 방법.
보다 큰 영역의 계산과 더 복잡한 연산을 위해서는 구조적인 융통성이 필요.
여러 개의 처리기들로 하나의 컴퓨터 시스템을 구성.

분산처리 방법.
하드웨어 자원이 프로세스 작업에 대하여 상대적으로 약하게 결합되어 동작.
컴퓨터 네트워크나 일부의 다중 컴퓨터가 해당.

병럴처리 방법.
하드웨어 자원들이 프로세스 작업에 대하여 강하게 결합되어 동작.
다중처리기와 다중 컴퓨터.

다중처리기.
독립적인 많은 처리기들이 하나의 컴퓨터 시스템을 구성.
하나의 운영체계를 가지면서 기억장치 공간의 입출력 자원을 공유.

다중 처리기의 구조.
처리기의 수가 증가함에 따라서 상호연결망은 전체 성능에 더 큰 영향을 미치게 된다.
가장 어려운 문제 중의 하나는 소프트웨어 부분.
MIMD 컴퓨터에 해당한다.
공유 기억장치 방식 : 전역적으로 물리적 주소 공간을 제공하여 프로그래밍의 편리함 제공.
메시지 전달 방식 : 설계는 쉽지만, 프로그램이( 소프트웨어 ) 어렵다.

다중컴퓨터.
자신만의 처리기, 기억장치, 입출력, 운영체제를 가진 여러 개의 컴퓨터로 구성.



 # 병렬처리기의 종류.

파이프라인 처리기.
프로그램 수행에 필요한 작업을 시간적으로 중첩하여 수행시키는 처리기.
파이프라인 처리원리를 명령어 수행에 적용시킨 것이다.

배열 처리기.
여러 개의 처리장치들이 배열 형태로 구성.

다중 처리기.
두 개 이상의 처리기가 동시에 수행할 수 있는 기능을 갖춘 컴퓨터 시스템.
프로그램의 수행속도를 포함한 성능의 개선이 주된 목표.
기억장치와 처리기 간 또는 기억장치와 입출력 채널 간의 상호 연결 구조.

데이터 흐름 컴퓨터.
프로그램에 내재한 병렬성을 최대한으로 이용하기 위한 구조적인 개선방법.
명령어의 순서에 무관하게 수행시키는 것.

VLSI 처리기.
병렬 알고리즘을 직접 하드웨어로 구현하는 새로운 처리기 구조.
대부분 파이프라인 기법을 이용한 다중처리 기법을 사용.
VLSI 연산장치는 규칙적인 단위 셀 등의 연결망으로 구성.



 # 상호연결망 구조.

상호연결망 interconnection network.
병럴처리시스템에서  처리 요소들과 기억장치들 사이를 연결하여 주는 네트워크.

상호연결망 관련 용어.
노드 : 네트워크의 상호연결에 대한 접합점.
네트워크 : 회선교환과 패킷교환 네트워크로 구분.
확장성 : 네트워크에 노드를 첨가함으로써 확장할 수 있는 특성.
중복성 : 메시지가 근원지에서 목적지로 전달 될 때 여러 가지 달느 경로를 지나는 것.
메시지 전달 네트워크 : 근원지에서 목적지로 정보를 쉽게 경로배정을 ㅅ할 수 있어야 한다.
네트워크 대역폭 : 단위시간에 전달 될 수 있는 데이터의 양.
연결도 : 한 노드를 떠나는 경로의 수.

상호연결망의 종류.
정적 상호연결망 : 직접 연결된 경로를 사용, 한번 마련된 경로를 변경할 수 없다.
동적 상호연결망 : 가능한 모든 유형의 통신을 할 수 있다. 범용적인 시스템 구축에 적합.

성형 구조.
허브라고 하는 특별히 할당된 노드와 연결되는 연결도 1의 노드들로 구성.
실제적으로 허브 이용에 대한 충돌때문에 십여 개의 노드로 제한된다.

선형 구조.
이용 가능한 가장 단순한 구조.
모든 노드는 연결도를 가지고 있기 때문에 노드를 추가하는 비용은 고정.

트리 구조.
최근의 많은 컴퓨터들에서 사용.
이진 트리 : 연결도는 일정하게 유지하면서 확장 가능한 구조라는 장점을 가진다.
이진 팻 트리 : 트리 구조의 문제점인 상위 수준에서의 병목현상을 해결할 수 있다.

메시 또는 토러스 구조.
처리기 배열에 적합.

하이퍼큐브 구조.
n차원 공간에서 정의되는 큐브의 2**n개의 꼭지점에 노드를 가진 구조.
병렬처리에서 매우 구조.
많은 처리기를 연결할 때도 좋은 효율.
높은 확장성을 가진다.
간단한 경로 배정 방법을 가진다.
모든 노드가 같은 경로 배정 방법을 사용할 수 있다.

단일 버스 구조.
일반적으로 미니컴퓨터나 마이크로컴퓨터에서 이용.
비용이 싸기 때문에 작은 규모의 다중처리기 시스템에 적합.
버스 사이클 당 하나의 연결만을 제공한다는 단점이 있다.

다중 버스 구조.
여러 개의 버스를 가지고 있는 구조.
여러 개의 장치가 다른 여러 개의 장치와 동시에 통신할 수 있기 때문에 고성능 시스템에 이용.

다단계 네트워크.
근원지와 목적지 사이에 여러 개의 스위치 요소가 있는 상호연결망.
보통 많은 연결이 필요하고 처리기의 수가 많을 때 이용.
연결망은 여러 단계로 구성되며, 인접한 두 단계는 순열연결을 이루게 된다.
