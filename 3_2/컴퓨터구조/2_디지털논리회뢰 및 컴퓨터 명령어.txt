
 디지털논리회로 및 컴퓨터 명령어.



 # 조합논리회로.

디지털 논리 회로 종류.
조합논리회로 combinational logic circuit : 현재의 입력에 의해서만 출력이 결정되는 논리회로.
순서논리회로 sequential logic circuit : 저장요소의 상태와 입력변수에 의해서 출력이 결정됨.

디지털 논리 회로 키워드.
분석 : 디지털 논리 회로가 어떻게 동작하는지 이해.
설계 : 내가 원하는 디지털 논리 회로를 설계 할 수 있는지.

가산기.
가산기 : 2진수의 덧셈을 수행하는 조합논리회로.
반가산기 : 두 비트의 덧셈을 수행. 
전가산기 : 세 비트의 덧셈을 수행. ( 1+1 통해 올림수가 발생했을 때 계산을 할 수 있다 )

감산기.
감산기 : 2진수의 뺄셈을 수행하는 조합논리회로.
반감산기 : 반가산기에 not 게이트를 추가한 것.
전감산기 : 전가산기에 not 게이트를 추가한 것.

가감산기.
덧셈과 뺄셈을 가산기만으로 수행하는 것이다. ( 회로를 최소화 ) 
감산기는 가산기에 인버터를 추가한 형태.



 # MSI를 이용한 조합논리회로.

디지털 시스템.
효과적인 조합논리회로를 설계하기 위해서 주어진 함수를 실현하는데 필요한 게이트의 수를 최소화.
이를 위해 직접회로( IC )를 사용.
이미 만들어진 MSI( Medium Scale Integrated circuits )장치를 이용하면 다양한 회로 설계가능.

MSI장치의 예.
인코더, 디코더, 멀티플렉서, 디멀티플렉서.

인코더 encoder.
부호화되지 않은 입력을 받아서 부호화된 출력을 내보내는 부호화기.
예 - 문자, 숫자, 기호 등을 2진 코드로 변환시켜 주는 조합논리회로.

디코더 decoder.
부호화된 입력을 받아서 부호화되지 않은 출력을 내보내는 복호화기.
n 비트의 2진 코드를 최대가 2**n개의 서로 다른 정보로 바꾸어 주는 조합논리회로.
2 X 4 디코더 : 2개의 입력이 있을 때 4개의 출력 중 하나만 1이 됨.
예 - 기억장치에서 특정 번지를 선택할 때나 컴퓨터 명령어를 해독하는 데 사용되는 조합논리회로.

멀티플렉서 mux multiplexer.
여러 개의 입력선 중에서 하나를 선택하여 단일의 출력을 내보내는 조합논리회로.
특정 입력선을 선택하기 위해서 선택변수를 사용. 
2**n개의 입력선 중에서 특정 입력선을 선택하기 위해서는 n개의 선택변수가 있어야 한다.
데이터 선택기라고도 한다.
4 X 1 멀티플렉서 : 4개의 입력이 있을 때 1개의 출력이 있음.

디멀티플렉서 demultiplexer.
데이터 분배기라고 한다.
1 X 4 디멀티플렉서 : 1개의 입력이 있을 때 4개의 출력이 있음.



 # 순서논리회로.

순서논리회로.
조합논리회로에 저장요소를 추가하는 것.

비동기 순서논리회로.
시간지연소자를 사용한다.
시간지연소자는 신호가 전달되려면 일정한 시간이 걸리므로 기억능력을 갖게 됨.
시간연소자는 게이트간의 피드백 때문에 예측 못한 상태의 동작이 발생됨.
이러한 이유로 자주 사용되지 않음.

동기 순서논리회로.
특정 순간에서만 저장요소의 내용을 변화시킬 수 있는 신호를 사용. ( 클럭 펄스 ) 
플립플롭이라는 저장요소를 사용하며, 가장 많이 사용되는 순서논리회로.

플립플롭 F/F Flip Flop.
한 비트의 2진 정보를 저장할 수 있는 장치.
입력신호에 의해 상태를 바꾸도록 지시가 있을 때까지 현재의 2진 상태를 유지하는 논리소자.
클럭 신호에 의해 출력상태를 바꾼다.

비동기 순서논리소자.
SR 래치 : 넓은 의미의 플립플롭, 구동방법이 다름 ( 구동입력이 1일 때 출력이 바뀜 ) 

동기 순서논리소자.
RS 플립플롭 : 세 입력이 모두 1일 때 어떤 값으로도 결정할 수 없는 상태 발생. ( R, S, C )
D 플립플롭 : RS플립플롭의 문제점을 보완. ( C,D - 1,1의 상황을 안 만듬 ) 
JK 플립플롭 : RS플립플롭의 문제점을 보완. ( C, J, K - 1,1,1일때 보수 ) 
T 플립플롭 : JK 플립플롭을 변형. ( 0은 무변화 , 1은 상태 반전 - Toggle )

레지스터.
데이터를 일시 저장하거나 전송하는 장치.
여러 개의 플립플롭을 연결하여 사용.
다양한 형태가 가능.

데이터 적재 레지스터.
입력된 데이터를 그대로 기억하는 역할을 수행하는 레지스터.
D F/F을 사용하여 구성.
직렬적재 레지스터 : 하나씩 F/F에 사용됨.
병렬적재 레지스터 : 한 번에 N개 F/F가 동시에 사용됨.
시프트 레지스터 : 레지스터에 입력돼있는 정보에 대해 원하는 방향으로 이동시킬 수 있는 레지스터.

카운터.
플립플롭을 사용해 만든 순서논리회로.
입력되는 클록 펄스의 적용에 따라 미리 정해진 순서를 밟아 가는 특수한 형태의 레지스터.
상향계수 카운터 : 수가 더해지는 카운터.
하향계수 카운터 : 수가 줄어드는 카운터.



 # 컴퓨터 명령어의 필요성.

디지털시스템의 분석.
디지털시스템은 디지털 논리회로가 모여진 것이다.
입력( 2진 신호 ) -> 디지털 논리회로 -> 출력( 2진 신호 )
입력( 컴퓨터 명령어 ) -> 컴퓨터 시스템 -> 출력( 명령어 수행결과 )

컴퓨터 명령어.
컴퓨터가 수행해야 하는 일을 나타내기 위한 비트들의 집합.
일정한 형식을 가짐.

명령어 집합.
컴퓨터에서 사용할 수 있는 명령어의 세트.
모든 컴퓨터는 자신의 명령어 집합을 가지고 있음.
명령어 집합은 그 컴퓨터의 구조적인 특성을 나타내는 가장 중요한 정보.
동일 계열의 컴퓨터는 같은 명령어 집합을 사용.
명령어 집합을 이용하여 컴퓨터 시스템의 구조를 살펴볼 수 있음.

명령어의 구성.
명령어는 필드라는 비트그룹으로 이루어지며, 연산코드와 오퍼랜드 필드로 구성.
연산코드 op-code : 처리해야 할 연산의 종류.
오퍼랜드 operand : 처리할 대상 데이터 또는 데이터의 주소.

컴퓨터 명령어의 수행 기능.
함수연산 기능, 정보전달 기능, 순서제어 기능, 입출력 기능.

명령어 형식.
명령어는 컴퓨터의 내부구조에 따라 여러 가지 형식이 있음.

명령어 형식의 분류.
오퍼랜드의 기억장소에 따른 명령어 형식.
오퍼랜드의 수에 따른 명령어 형식.
