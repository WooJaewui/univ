

 1. 수업에서 JSP 웹 애플리케이션 개발환경의 웹서버와 웹응용 서버로 사용한 서버 이름은 무엇인가?
    톰캣 Tomcat.

 2. 수업에서 JSP 웹 애플리케이션 개발환경으로 설치하고 사용한 JSP 통합개발환경은 무엇인가 ?
    이클립스 eclipse.

 3. 웹 애플리케이션을 실행하고 관리하는 별도의 전담 프로그램으로 웹 페이지 생성 외에도 
    API 제공, 부하 균형, 고장 조치 등을 수행하는 서버를 영어(또는 영어 약자)로 무엇이라 하는가?
    WAS Web Application Server.

 4. JSP의 스크립트 요소에서 JSP 페이지에 Java 코드를 삽입되도록 하는 스크립트 요소 이름은 무엇인가?
    표현식 Expression.

 5. page 지시어 중 버퍼가 차면 자동으로 버퍼를 비워 내용을 클라이언트에 보낼 여부를 지정하는 속성은 무엇인가?
    autoFlush.

 6. JSP의 스크립트 요소에서 JSP 페이지에 변수 선언 또는 메소드 선언문을 삽입하도록 하는 스크립트 요소 이름은 무엇인가?
    선언 Declaration.

 7. JSP 페이지의 구성 요소 중의 하나로, 자주 사용되는 기능을 
    JSP 컨테이너가 묵시적으로 이미 정의되어있는 변수 객체 형태로 제공한 것을 무엇이라 하는가?
    내장 객체.
