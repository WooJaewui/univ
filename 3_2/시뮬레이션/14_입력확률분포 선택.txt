 # 확률 분포의 선택.

발견적기법 heuristics.
연속형 분포 : 점통계량, 히스토그램, 확률도.
이산형 분포 : 점통계량, 막대그래프.

점통계량 point statistics.
분산계수를 사용.
쉽고 빠르나 통계적 평가 등이 없어 신뢰성 문제.

히스토그램.
밀도함수의 형태를 추정.
구간선정과 넓이 선정이 어렵다. ( 이산형 - 막대그래프 )

확률도 probability plots.
예측한 분포를 가정해서 확률도를 그리면 직선 형태.
미리 적합한 분포를 정하고, 형태 모수를 예측해야하며 역함수를 구할 수 없는 경우도 있다.
이산형은 적용 불가.

통계적 기법에 의한 육안 평가.
도수분포표.
막대그래프( 히스토그램 )
평균과 분산.

적합도 검정.
chi-square( x**2 ) text.
Kolmogorov - Smironov test.



 # 적합도 검정.

설명.
관측치 X1, X2, ..., Xi들이 가정한 분포 F에 적합한가를 검정.
귀무가설 H0 : 주어진 Xi들은 F를 가진 확률변수이다.

검정방법.
평균과 분산은 알고 있어야 함.

chi-square( x **2 ) test.
이산형 및 연속형 모두 가능.
n -> 무한대이면 적용이 곤란.

Kolmogorov - Smironov test.
이론적 확률분포로부터 얻은 누적확률분포와 관측자료로부터 얻은 누적 확률 분포의 차를 비교하여
가장 큰 D를 구한다. ( D - 통계량이 K-S기각점보다 크면 귀무가설을 기각할 수 없다 )



 # 시간당 전화문의 예시.

특징.
연속된 시간 구간에서 발생된 이산사건.
한 사건의 출현이 다른 사건의 발생에 전혀 영향을 미치지 못함.
0의 발생확률이 매우 높다.
구간별 평균 발생이 매우 작다. ( 0.5147 )
포아송 분포로 가정.



 # x ** 2 검정.

귀무가설 : H0.
관측치의 분포와 가정한 이론분포와 유의한 차이가 없다.
유의수준 0.05.
x ** 2 기각점과 x ** 2 통계량을 비교.

요구사항.
실제 빈도수나 숫자만 사용된다.
각 구간에 대한 기대빈도는 5이상.
자유도 v계산 : v = k - 1 - m ( K = 구간의수 , m = 매개변수의 숫자 )



 # Kolmogorov - Smirnov 검정 ( K-S 검정 )

설명.
이론적 확률분포로부터 얻은 누적값 확률분포와 관측자료로부터 얻은 누적값 확률분포를 비교.
각 구간에서 이론치와 관측치의 절대값 차이가 가장 큰 값을 찾음.
K-S 분포표에서 구한 값과 비교, 자유도=N( 표본크기 )

누적값 계산.
이전까지의 값의 관측 확률 + 어떤 값의 관측 확률 = 관측 확률 누적 값. ( 모두 더하면 1 ) 
이전까지의 값의 이론 확률 + 어떤 값의 이론 확률 = 이론 확률 누적 값. ( 모두 더하면 1 )
D = 차이.



 # 검정방법 비교.

x ** 2 검정.
표본의 크기가 큰 경우 ( 100 <= n )

K-S 검정.
표본의 크기가 작은 경우 ( 10 <= n <= 99 )

Cramer-Von Mises 검정.
표본의 크기가 아주 작은 경우 ( n <= 10 )
