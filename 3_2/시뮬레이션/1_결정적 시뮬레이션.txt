 # 시뮬레이션 개념.

시뮬레이션 정의.
실제 시스템( Real System )에서 실제 시스템의 성질을 나타내는 모델을 만들고 그 모델로 실험하는 것.

기본요소.
실제시스템 -> 모델링 -> 모델 -> 시뮬레이션 -> 시뮬레이터.

목적.
실제 시스템이 갖는 행위를 예측, 평가, 분석, 훈련.

시뮬레이션 특징.
미래를 예측한다. ( 그림 사용 )
시간변수 사용.
다양한 실험이 가능하다.

시뮬레이션의 구분.
결정적( 확정적 ) 시뮬레이션 deterministic simulation : 확률 변수가 없다.
확률적 시뮬레이션 probabilistic simulation : 확률 변수가 있다.

시뮬레이션 결과.
시뮬레이션은 결과를 보기 쉽게 하기 위해 표나 ppt 등을 통해 정리하는 것이 좋다.

시뮬레이션 활용 예.
DRIVING 시뮬레이션 : 실내에서 운전연습을 할 수 있다.
사이버 모델하우스 : 앞으로 세워질 주거공간을 미리 체험해 본다.
의료영상 시뮬레이션 : 보다 정확한 의료진단을 위해 적용되고 있다.
시뮬레이션 게임 : 상상을 현실로 만들어 볼 수 있다.
교육용 시뮬레이션 : 물리, 수학, 공학 등 까다로운 원리를 흥미롭게 학습할 수 있다.

결정적 시뮬레이션 예.
연봉 1200만원 : salary.
매년 5% 이상 : salary = salary + 0.05 * salary.
저축액 매년 연봉의 10% : saving = savings + 0.10 * salary.
은행이율 매년 복리 5% : saving = saving + 0.05 * saving.
