 # 기본문제. 

이전 예제 조건.
도착상태 : 평균 15대/H. ( 일양분포 )
봉사시간 : 평균 4분. ( poisson 분포 )
주유대 : 1대.
시간 : 100분.

도착상태의 변화.
도착상태 : 평균 20대/H. ( 일양분포 )
봉사시간 : 평균 4분. ( poisson 분포 ) 
주유대 : 1대.
시간 : 100분.

복수창구의 대기행렬.
도착상태 : 평균 20대/H. ( 일양분포 )
봉사시간 : 평균 4분. ( poisson 분포 )
주유대 : 2대.

시뮬레이션 시간.
어느정도 많은 시간동안 시뮬레이션하면 시간을 늘려도 값이 크게 변하지 않는 안정상태가 발생하는데,
안정상태까지 시뮬레이션을 실행하는 것이 좋다.

알고리즘 초기값.
queue = 0 ;
totque = 0 ;
totarr = 0 ;
tpump = 0 ; ( 배열 - 주유대 2대 ) 
time = 0 ;
tlimit = 100 ;
tstep = 1 ;

알고리즘 결과값.
평균대기시간 = totquq( 총대기시간 ) / tlimit( 시뮬레이션시간 ) * tstep( 구간을 나누는 시간 ) ) 
평균대기시간을 계산할때 tlimit와 tstep의 시간단위를 맞춰서 계산해야된다. ( 시, 분, 초 )
