 # 확률적 시뮬레이션.

확률적 시뮬레이션 probabilistic simulation = stochastic simulation.
확률변수( random variable )을 사용하는 시뮬레이션이다.

확률변수 Random variable.
상황에 따라 다른 확률로 값을 가지게 되는 변수이다.
일양분포, 정규 분포, 지수 분포가 있다.

값의 분포에 따른 분류.
이산적 분포 : 변수 값이 불연속인 값을 가지는 경우. ( 예 - 주사위 )
연속적 분포 : 변수 값이 정해진 모든 구간의 값을 연속적으로 취하는 일.

일양분포 Uniform distribution.
변수의 모든 값에서 일정한 확률을 가진다. ( 주사위 던지기 )

정규분포.
평균 점수 m을 기준으로 m이 가장 확률이 높고, 오른쪽과 왼쪽으로 갈수록 점차 확률이 줄어드는 분포도.

지수분포.
x = 0일 때 확률이 가장 높고, 값이 커질수록 확률이 줄어든다.

포아송 분포. poisson distribution.
출생자수처럼 연속된 구간에서 독립적인 사건이 일어난다. 

난수 Random number
특정한 순서나 규칙을 가지지 않는 수를 의미한다. 
발생 확률이 같아서 예측을 할 수 없다.

그래프 포함관계.
[ : 포함시킨다. ( 이상 또는 이하 )
( : 포함시키지 않는다. ( 초과 또는 미만 )
예 - [0,1) : 0이상 1미만.

몬테칼로 simulation.
반복된 무작위 추출을 이용하여 함수의 값을 확률적으로 계산하는 알고리즘.
