
 선택 제어문과 반복 제어문.



 # 선택 제어문.

프로그램 언어의 제어 구조.
● 순차적 제어 : 특별한 지정이 없는 한 위에서 아래로 수행되는 제어구조.
● 선택적 제어 : 주어진 조건에 따라 특정부분으로 수행을 옮긴느 분기 제어구조.
● 반복적 제어 : 특정 부분을 일정한 횟수만큼 반복 수행하는 반복 제어구조.

선택 제어문의 종류.
● if문.
● switch ~ case문.
● goto문.

*** 단순 if문 예시 ***
if( 조건 ) { 
	조건이 참이면 실행할 문장 ;
}

*** if ~ else 문 ***
if( 조건 ) { 
	조건이 참이면 실행할 문장;
} else {
	조건이 거짓이면 실행할 문장;
}

*** 다중 if ~ else 문 ***
if( 조건1 ) { 
	if( 조건2 ) {
		조건1,2가 모두 참일 때 실행할 문장;
	} else { 
		조건1이 참이고, 조건2가 거짓일 때 실행할 문장;
	}
} else {
	조건1,2가 모두 거짓이면 실행할 문장;
}

*** if ~ else if ~ else 문 ***
if( 조건1 ) {
	조건1이 참일 때 실행할 문장 ;
} else if( 조건2 ) { 
	조건1이 거짓이고, 조건2가 참일 때 실행할 문장 ;
} ... { 
	
} else { 
	조건이 모두 거짓일 때 실행할 문장 ;
}

*** switch ~ case 문 ***
switch( 수식 ) { 
	case 값1: 수식이 값1일 때 실행할 문장;
		break ;
	case 값2: 수식이 값2일 때 실행할 문장;
		break ;
	...
	default : 수식이 모든 값과 맞지 않는 경우 실행할 문장 ;
}		

*** go to 문 ***
Label: 
goto Label ; ( Label 위치로 무조건 이동 )

go to문 주의사항.
{ } 안으로는 들어갈 수 없다.
{ } 밖으로는 나갈 수 있다.



 # 반복 제어문.

반복 제어문의 종류.
● for문.
● while문.
● do ~ while 문.

*** for문 ***
for( 초기값 ; 조건문 ; 증감식 ) { 
	조건이 false가 될때까지 반복할 문장 ;
}

*** 다중 for문 ***
for( 초기값 ; 조건문 ; 증감식 ) { 
	for( 초기값 ; 조건문 ; 증감식 ) { 
		...
	} 
	...
}

*** while문 *** 
while( 조건 ) { 
	조건이 false가 될때까지 반복할 문장 ;
}

*** do ~ while문 ***
do { 
	조건이 false일 때까지 반복할 문장; ( 조건에 상관없이 무조건 1번 실행 )
} while( 조건식 ) 



 # 기타 제어문.

기타 제어문의 종류.
● break문.
● continue문.

break문.
● 반복 명령의 실행 도중에 강제적으로 반복문을 빠져 나오는데 사용.
● for, while, do ~ while, switch 블록 등을 빠져나옴.

continue문.
● 루프 실행 중에 루프를 다시 실행하고자 할 때 사용.
● 반복구조에서만 사용 가능.
