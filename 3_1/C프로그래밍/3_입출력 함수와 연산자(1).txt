 # 표준 입출력 함수.

함수란 무엇인가?
함수란 특정한 작업을 수행하도록 설계됙 독립적인 프로그램.

C언어에서의 함수.
● 표준함수 : C언어 자체에서 제공하는 함수.
● 사용자 정의 함수 : 사용자가 정의하여 사용하는 함수.

표준 입출력 함수.
선행처리기.
	● #include <stdio.h>
표준 출력함수.
	● printf( ) : 화면에 여러 종류의 자료를 출력.
	● putchar( ) : 화면에 1개의 문자를 출력.
	● puts( ) : 화면에 문자열을 출력.
표준 입력함수.
	● scanf( ) : 키보드를 통해 여러 종류의 자료를 입력 받음.
	● getchar( ) : 키보드를 통해 1개의 문자를 입력 받음.
	● gets( ) : 키보드를 통해 문자열을 입력 받음.

출력양식 변환기호.
● %d : 부호 있는 10진 정수로 변환하여 출력. ( 정수형, 문자형 )
● %u : 부호 없는 10진 정수로 변환하여 출력. ( 정수형, 문자형 )
● %f : 부동소수점형식으로 변환하여 출력. ( 실수형 )
● %e, %E : 지수형태로 변환하여 출력. ( 실수형 )
● %c : 한 문자로 변환하여 출력. ( 정수형, 문자형 )
● %s : 문자열로 변환하여 출력. ( 문자열 포인터 )

출력양식의 편집.
● %5d : 총 5자리로 오른쪽부터 채워짐.
● %-5d : 총 5자리로 왼쪽부터 채워짐.

입력양식 변환기호.
● %d : 부호 있는 10진 정수로 입력.
● %u : 부호 없는 10진 정수로 입력.
● %f : 부동소수점형식으로 입력.
● %e, %E : 지수형태로 입력.
● %c : 한 문자로 변환하여 입력.
● %s : 문자열로 변환하여 입력.

#pragma warning( diable:4996 )
권장하지 않는 함수 사용에 대한 메시지 무시.
scanf_s( ) 를 사용하면 경고 메시지가 발생하지 않는다.

gets( )와 scanf( ) 문자열 입력 차이.
gets( )로 컴퓨터 과학과를 입력받으면 컴퓨터 과학과를 출력. ( 공백도 문자열로 인식한다 )
scanf( )로 컴퓨터 과학과를 입력받으면 컴퓨터를 출력. ( 이스케이프 문자를 통해 문자를 구분한다 )

puts( )와 printf( ) 문자열 출력 차이.
puts( )는 문자가 끝나면 \n을 통해 줄바꿈을 해준다.
printf( )는 문자가 끝나도 줄바꿈을 하지 않는다.
