
 자바스크립트 기본문법.



 # 개념과 선언 방식.

JavaScript.
웹페이지의 동작을 담당.
Live Script가 발전되어 JavaScript가 되었다.

스크립트.
컴퓨터 프로세서가 아닌 다른 프로그램에 의해 번역되거나 수행되는 프로그램이나 명령어 나열.

자바스크립트의 특징.
HTML 문서 내에 삽입된다. ( 운영체제 / 하드웨어의 환경에 독립적 )
웹 브라우저에 의해 해석 / 실행. ( 빠른 개발 및 결과 확인 용이 )
스크립트 언어. ( 이해하고 배우기 쉬움 )
객체 기반. ( 자바와는 달리 제한적인 객체지향 개념 도입 )
웹 브라우저에서 소스 보기가 가능. ( 취약한 보안 )

자바스크립트 선언 방식 - 내부 스크립트 방식.
<script>
	자바스크립트 코드 ;
</script>

자바스크립트 선언 방식 - 외부 스크립트 방식.
<script src="파일이름.js"></script>

자바스크립트 선언 방식 - 인라인 스크립트 방식.
<div onClick="함수이름( )">
	...
</div>

자바스크립트 주석.
// : 한 줄 주석.
/* ... */ : 여러 줄 주석.



 # 변수, 자료형, 연산자.

변수.
프로그램에서 다루는 데이터를 저장하기 위한 공간.

변수 선언.
데이터를 저장할 적절한 크기의 공간을 확보하는 것.
자료형에 상관없이 var를 통해 선언 가능.

식별자.
변수/함수의 이름으로 사용하기 위해 프로그래머가 만든 단어.

식별자 지정 규칙.
첫 글자는 반드시 영문자 , '_' , 또는 '$'로 시작.
두 번째 글자부터는 영문자 , 숫자 , '_'의 조합으로 구성.
대소문자 구분 , 키워드 사용 불가.

키워드. ( 예약어 )
특별한 목적으로 사용하기 위해서 미리 정해 놓은 단어.

자료형.
10진수 : 25 , 123 , ...
8진수 : 025 , 0321 , ...
16진수 : 0x25 , 0x6af , ...
실수 : 123.00 , 1.23e2 , ...
문자열 : "hello" , ...
논리형 : true , false.
null형 : null.

연산자.
산술 연산자 : + , - , * , / , %.
대입 연산자 : = , += , -= , *= , ...
증감 연산자 : ++ , --
비교 연산자 : == , >= , != , <= , ...
논리 연산자 : ! , && , ||.
삼항 연산자 : a > b ? a : b ;



 # 제어문.

조건문.
프로그램의 여러 실행 경로 중에서 조건식의 참/거짓 여부에 따라 하나의 실행 경로를 선택하는 문장.

if-else 문.
if( 조건문1 ) { 
	조건문1이 true일 때 실행할 문장 ;
} else if ( 조건문2 ) { 
	조건문2가 true일 때 실행할 문장 ;
} else { 
	조건문1 , 조건문2가 false일 때 실행할 문장 ;
}

switch문.
switch( 변수 ) { 
	case 값1 : 변수가 값1일 때 실행할 문장 ;
		 break ;
	case 값2 : 변수가 값2일 때 실행할 문장 ;
		 break ;
	...
}

for문.
for( 초기값 , 조건 , 증감식 ) { 
	조건이 true일 때 반복할 문장 ;
}

while문.
while( 조건 ) { 
	조건이 true일 때 반복할 문장 ;
}

do~while문
do { 
	1번 무조건 실행되고 , 조건식이 true일 때 반복할 문장 ;
} while ( 조건식 ) 

continue문.
반복 도중에 continue문을 만나면 이후의 반복 문장들을 건너뛰고 반복문의 시작 부분으로 돌아감.

break문.
반복 수행을 종료.



 # 함수.

함수.
프로그램 내에서 논리적으로 연관된 일련의 식/문장 등을 하나의 단위로 묶어 놓은 집합.
프로그램의 구조화, 코드의 길이가 짧고 유지보수 등이 용이.

익명 함수. ( 무명 함수 )
함수명을 지정하지 않고 선언된 함수.
함수를 변수에 저장하였다가 변수를 통해서 사용하는 경우.

콜백 함수.
매개변수로 전달되는 함수.

이미 만들어진 함수.
alert( ) : 확인 버튼을 갖고 메시지 창을 띄우는 함수. ( 경고 안내 메시지 )
confirm( ) : 사용자에게 메시지를 보여주고 확인/취소 버튼을 선택하게 함.
prompt( ) : 사용자로부터 키보드로 데이터를 입력받을 때 사용.
