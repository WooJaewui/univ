 # CSS 사용.

CSS Cascading Style Sheets 란 ? 
웹 문서에 글꼴, 색상, 여백과 같은 스타일을 추가하는 간단한 메커니즘.

CSS 사용의 장점.
HTML의 단순한 기능에 대해 CSS를 통해 기능의 추가/변경.
한 번의 속성 정의로 여러 요소/문서에 적용 가능.
웹 문서의 내용과 스타일 정보의 분리가 가능.
통일된 문서 양식의 사용을 통해 처리할 데이터 양의 감소.

CSS 정의 형식.
선택자 { 속성 : 값 ; } 
선택자 selector : 스타일을 적용하려는 HTML의 요소.
선언 declaration : 속성 + 값.
속성 property : 지정한 선택자의 어떤 부분에 대해서 스타일을 적용할 것인가를 지정하는 부분.
값 value : 해당 속성에 지정할 스타일의 값.

CSS 적용 방법.
인라인 방법 : 태그안에서 CSS를 속성안에서 적용하는 방법.
임베디드 방법 : HTML 문서 안에 CSS를 작성하는 방법.
링크 방법 : CSS파일을 만들어놓고 HTML에서 적용시키는 방법. ( W3C 권장방법 ) 

CSS 적용의 우선순위.
적용 태그에 가까운 순서대로 우선순위를 갖는다.



 # CSS 선택자.

기본 선택자의 종류.
 * : 모든 요소에 스타일 적용.
요소명 : 지정한 요소에만 스타일 적용.
.클래스명 : 지정한 클래스명을 사용한 요소에만 적용.
#아이디명 : 지정한 아이디명을 사용한 요소에만 적용.
요소명[속성] : 지정한 속성명을 사용한 요소에만 적용.



 # 의사 클래스, 결합자, 의사 요소.

의사 클래스 pseudo class.
요소에는 실제로 존재하지 않지만 클래스가 선언된 것처럼 간주하여 만즌 선택자.
사용방법 - '기본선택자 : 의사클래스'
링크 의사 클래스 : link , visited.
사용자 동작 의사 클래스 : hover, active, focus.

결합자 cominator.
선택자를 문맥이나 요소의 구조를 기반으로 결합하는 방법.

종류
자손 결합자 : 선택자1 선택자2.
자식 결합자 : 선택자1>선택자2.
인접 형제 결합자 : 선택자1+선택자2.
일반 형제 결합자 : 선택자1~선택자2.
그룹 결합자 : 선택자1, 선택자2, ...

의사 요소.
HTML에서 접근할 수 없는 정보와 소스 문서에 존재하지 않는 컨텐츠에 대한 참조가 가능한 수단.
사용 방법 - '기본선택자 : 의사요소'

의사 요소.
::first-line : 특정 요소를 사용하는 문단의 첫 번째 줄의 텍스트를 선택.
::first-letter : 첫 번째 줄의 텍스트에서 첫 번째 글자를 선택.
::before : 요소 내용의 맨 앞에 추가되는 내용과 스타일 지정.
::after : 요소 내용의 맨 뒤에 추가되는 내용과 스타일 지정.
