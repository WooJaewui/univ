
 jQuery(2)



 # 요소 및 속성 조작.

요소의 내용 및 속성 조작 메소드.
text( ) : 요소의 텍스트 컨텐츠를 가져오거나 수정할 때 사용.
html( ) : 요소의 HTML 태그가 포함된 텍스트 컨텐츠를 가져오거나 수정할 때 사용.
val( ) : 입력양식에서 사용자가 입력/선택한 값을 가져올 때 사용.
attr( ) : 요소의 속성값을 가져오거나 기존 값을 수정할 때 사용.

attr( ) 메소드 속성값 설정하기.
$( 선택자 ).attr( 속성명 , 값 ) ;
$( 선택자 ).attr( { 속성명1 : 값1 , 속성명2 : 값2 , ... } ) ;

메소드 체이닝.
동일한 요소에 대해서 여러 메소드를 연속적으로 적용.
메소드를 연결하여 하나의 문장으로 만들어서 사용.

removeAttr( ) 메소드.
선택된 요소에서 하나 이상의 속성 삭제.
$( 선택자 ).removeAttr( 속성명 ) ;



 # CSS 조작.

css( ) 메소드.
스타일의 속성값을 가져오거나 요소의 스타일 지정.
 - 스타일을 가져오기.
	- $( 선택자 ).css( 속성명 ) : 
 - 스타일 지정하기.
	- $( 선택자 ).css( 속성명, 값 ) ;
	- $( 선택자 ).css( { 속성명1 : 값1 , 속성명2 : 값2 , ... ) ;

hover 이벤트.
마우스오버 이벤트와 마우스리브 이벤트가 같이 연결되어 있다.
함수 2개를 선언해서 마우스오버, 마우스리브 이벤트에 대한 코드를 실행시킬 수 있다.

class 속성을 편집하기 위한 메소드.
addClass( ) : 선택된 요소에 하나 이상의 클래스명 추가.
removeClass( ) : 선택된 요소로부터 하나 이상의 클래스명 삭제.
toggleClass( ) : 선택된 요소로부터 클래스명의 추가와 삭제를 번갈아 가면서 수행.



 # DOM 요소 편집.

요소 추가.
$( 선택자 ).append( 컨텐츠 ) : 선택된 기존 요소 안에서 끝부분에 새로운 컨텐츠 삽입.
$( 선택자 ).prepend( 컨텐츠 ) : 선택된 기존 요소 안에서 앞부분에 새로운 컨텐츠 삽입.
$( 선택자 ).before( 컨텐츠 ) : 선택된 요소의 앞에 새로운 컨텐츠 삽입.
$( 선택자 ).after( 컨텐츠 ) : 선택된 요소의 뒤에 새로운 컨텐츠 삽입.

요소 추가 예시 코드.
<script>
	$( document ).ready( function( ) {  
		$( "#btn1" ).click( function( ) { 
			$( "div" ).prepend( "<b>먼저 작성될 내용</b>" ) ;
			$( "div" ).append( "<b>나중에 작성될 내용</b>" ) ;		
		}) ;
	}) ;
</script>

기존 요소의 삭제.
remove( ) : 선택된 요소와 자식 요소를 포함해서 모든 것을 삭제.
empty( ) : 선택된 요소만 빈 요소로 남기고, 그 안에 포함된 모든 자식 요소 삭제.



 # jQuery 효과.

Hide / Show.
hide( ) : 선택된 요소를 화면에서 보이지 않도록 감춤.
show( ) 선택된 요소를 숨겨진 상태에서 화면에 표시.
toggle( ) : hide( )와 show( )를 번갈아 수행.

페이드.
fadeIn( ) : 감춰진 요소를 점차적으로 밝게 만들어서 화면에 표시.
fadeOut( ) : 요소를 점차적으로 흐리게 만들어 화면에서 사라지게 함.
fadeToggle( ) : fadeIn( )과 fadeOut( )을 번갈아 수행.
fadeTo( ) : 지정한 불투명도까지의 페이드아웃 효과를 부여.

슬라이드.
slideUp( ) : 선택된 요소를 위쪽으로 밀어 올려 보이지 않도록 함.
slideDown( ) : 감춰져 있던 요소를 아래쪽으로 밀어 내려서 화면에 표시.
slideToggle( ) : slideUp( )과 slideDown( )을 번갈아 수행.

애니메이션.
animate( ) : 사용자가 원하는 애니메이션 효과를 지정.

hide / show 예시 코드.
$( 선택자 ).hide( 속도 ,콜백함수 ) ;
$( 선택자 ).show( 속도 , 콜백함수 ) ;
$( 선택자 ).toggle( 속도 , 콜백함수 ) ;

페이드 예시 코드.
$( 선택자 ).fadeIn( 속도 , 콜백함수 ) ;
$( 선택자 ).fadeOut( 속도 , 콜백함수 ) ;
$( 선택자 ).fadeToggle( 속도 , 콜백함수 ) ;
$( 선택자 ).fadeTo( 속도 , 불투명도 , 콜백함수 ) ; 
  - 불투명도는 0.0 ~ 1.0.

슬라이드 예시 코드.
$( 선택자 ).slideUp( 속도 , 콜백함수 ) ;
$( 선택자 ).slideDown( 속도 , 콜백함수 ) ;
$( 선택자 ).slideToggle( 속도 , 콜백함수 ) ;

애니메이션 예시 코드.
$( 선택자 ).animate( { CSS속성 } , 속도 , 콜백함수 ) ;
CSS 속성은 필수 지정, 속도/콜백함수는 생략 가능.

애니메이션 설정 주의사항.
속성값이 숫자로 지정되는 경우에만 CSS속성으로 지정 가능하다.
