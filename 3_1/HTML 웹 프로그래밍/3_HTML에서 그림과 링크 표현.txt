
 HTML에서 그림과 링크 표현.



 # 이미지 삽입.

<img> 태그.
종료태그 없이 사용.
src : 문서에 표시할 이미지 파일을 지정.
width : 이미지의 폭 지정.
height : 이미지의 높이 지정.
alt : 이미지가 표시되지 않을 때 대체될 텍스트 지정.
usemap : 클라이언트 측의 이미지맵 지정.

src 속성.
HTML 문서에 표시할 이미지 파일의 이름/경로 지정. ( 필수 속성 )

GIF Graphics Interchange Format.
256색상 지원.
투명 배경 처리.
용량이 작음.
로고나 클립아트 형태의 이미지에 적합.

JPG Joint Photograhpic coading experts Groups.
트루컬러 지원.
좋은 압축률 -> 용량이 작음.
사진 이미지에 적합.

PNG Portable Network Graphics.
트루컬러 지원.
투명 배경 처리.
웹 전용 이미지 파일 ( 빠른 화면 표시 속도 )
클립아트와 같이 적은 색상을 가진 이미지에 적합.

width , height 속성.
이미지의 크기 지정.
기본 출력 크기 -> 이미지의 원본 크기.
속성값 -> 픽셀 , %( 브라우저 화면 크기에 대한 상대적 크기 )
둘 중 하나의 속성의 값만 지정하면 다른 값은 원본 크기에 비례해서 자동 지정.

alt 속성.
이미지에 대한 대체 텍스트를 지정.



 # 하이퍼링크 삽입.

<a> 태그.
href : 링크를 걸어 줄 내용물( 텍스트 , 이미지 )

href 속성.
링크를 통해 이동하려는 곳의 경로/주소 지정.
#html , .html 등을 통해 id , class명으로 이동할 수 있다.
javascript JS를 실행할 수도 있다.

target 속성.
링크가 클릭되었을 때 링크로 연결된 내용이 표시될 창을 지정.
_blank : 새로운 창에서 연다.
_self : 현재의 창에서 연다. ( 기본값 )
_top : 현재의 창이 여러 프레임인 경우 하나의 화면에서 문서를 염.
_parent : 부모 프레임에서 문서를 연다.
프레임명 : 프레임 이름으로 연다.

download 속성.
href 속성에 의해 지정된 파일을 직접 다운로드.

title 속성.
마우스 오버했을 때 나타날 글자를 표시.



 # <iframe> 태그와 이미지맵.

<iframe> 태그
현재의 페이지 안에서 다른 웹페이지를 표시하는 창을 삽입하기 위한 태그.
name : 창의 이름 지정.
src : 창에 표시될 문서의 URL 지정.
srcdoc : 창에서 보여줄 HTML 컨텐츠 지정. ( srcdoc을 지정하면 src를 무시함 )
width : 창의 폭 지정.
height : 창의 높이 지정.

이미지맵.
하나의 이미지를 여러 개의 영역으로 나누어 각 영역마다 링크를 지정하는 것.
<img src="url" usemap="#맵이름">
<map name="맵이름">
	<area .... >
</map>

<area> 태그.
shape : 이미지맵에서 각 영역의 모양.
coords : 각 영역을 구성하는 좌표값.
href : 이미지 링크.
