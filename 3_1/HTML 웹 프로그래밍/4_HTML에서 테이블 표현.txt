
 HTML에서 테이블 표현.



 # 테이블의 기본 형식.

테이블 만드는 방법.
<table>
	<tr>
		<td> 1행 1열 </td>
		<td> 1행 2열 </td>
		..
	</tr>
	<tr>
		..
	</tr>
</table>

<table> 태그.
HTML에서 테이블을 정의하는 태그.

<tr> 태그.
테이블에서 하나의 줄에 해당하는 행( row )을  정의한느 태그.

<td> 태그.
테이블에서 데이터가 표시되는 표준 셀의 정의.
셀 내용 -> 왼쪽 정렬로 표시.

<th> 태그.
테이블 내부에서 행/열의 제목을 표시하는 헤더 셀을 정의.
사용하는 경우에는 반드시 <tr> 태그 내에서 사용.
셀 내용 -> 진하게 가운데 정렬로 표시.

<caption> 태그.
테이블 전체에 해당하는 제목을 표시.
테이블 레이아웃에는 아무런 영향을 미치지 않음.
반드시 <table> 시작태그의 바로 다음에 위치해야 함.
테이블 상단에 진하게 가운데 정렬로 표시.



 # 셀 병합과 요소 배치.

셀 병합.
인접한 여러 셀을 합쳐서 하나의 셀로 만드는 경우.
보다 다양한 모양의 테이블 생성이 가능.
<td> , <th> 태그 내에서 rowspan 속성, colspan 속성을 통해 병합한다.
rowspan = n : 현재 셀의 위치에서 n개의 행을 병합. ( 세로 방향으로 n개 병합 )
colspan = n : 현재 셀의 위치에서 n개의 열을 병합. ( 가로 방향으로 n개 병합 )

테이블을 이용한 요소 배치.
바둑판식 배열처럼 질서 정연한 요소의 배치가 가능하다.
각 셀의 내용으로 텍스트, 이미지, 목록, 테이블 등이 가능하다.



 # 테이블 콘텐츠 그룹핑과 열 단위 스타일 지정.

행 단위 컨텐츠의 그룹핑.
<thead> , <tbody> , <tfoot> 태그.
테이블에서 행 단위 컨텐츠를 각각 헤더, 몸체, 푸터 부분으로 구분하여 그룹핑하는 태그.
테이블의 레이아웃에는 아무런 영향을 미치지 않음.
<thead> , <tbody> , <tfoot>의 순서대로 사용.

<colgroup> 태그.
하나 이상의 열들을 모아서 한꺼번에 스타일을 지정하는 경우.

<col> 태그.
지정한 열에 대해서 서로 달느 포맷팅을 정의하는 경우.
<colgroup> 태그 내에서만 사용.
span = n : n 개의 열을 묶어서 표현.
종료태그 없이 사용.
