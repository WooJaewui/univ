
 HTML에서 멀티미디어 표현.



 # audio 요소.

<embed> 태그.
외부 애플리케이션이나 상호작용 컨텐츠를 위한 컨테이너.
HTML5에서 새롭게 추가. ( HTML4에서는 미지원 )
src  : 내장되는 외부 파일의 주소.
type : 내장되는 컨텐츠의 미디어타입.
width : 폭.
height : 높이.

<audio> 태그.
외부 플러그인의 도움 없이 간단히 오디오 재생.
<audio src="오디오 파일 주소">
	오디오 파일이 실행되지 않을때 수행할 코드 ;
</audio>
src : 재생하라 오디오 파일의 URL.
autoplay : 웹 문서 로딩 시에 오디오를 자동으로 재생할 지 여부.
controls : 오디오 제어기( 재생, 일시정지, 볼륨 등 )의 표시 여부.
loop : 오디오의 반복 재생 여부.
muted : 음소거 여부.
preload : 웹 문서가 로딩될 때 오디오의 로딩 상태. ( none / auto( 기본값 ) / metadata )

웹에서 주로 사용되는 오디오 형식.
.mp3 : MPEG-1의 오디오 규칙으로 개발된 오디오 압축 기술.
.wav : 윈도우에서 오디오 재생을 위해 MS와 IBM이 개발한 비압축 방식.
.ogg / .oga : MP3 대안으로 개발되어 스트리밍 방식의 멀티미디어 표현을 위한 공개 소스 기반의 파일 형식.

브라우저별 지원 오디오 파일 형식.
각 브라우저/버전마다 지원되는 파일 형식이 다름.

<source> 태그.
하나의 소스 파일에 대해 서로 다른 형식을 갖는 여러 파일을 동시에 지정하기 위한 태그.
<audio>와 <video> 태그의 src 속성의 역할을 대신함.
<audio>와 <video> 태그의 내에서 여러 개 지정. ( 가장 위의 <source>부터 재생 )
src : 재생할 미디어 파일의 URL.
type : 재생할 파일의 MIME 형식. ( audio/mp3 , audio/wav , audio/ogg )



 # video 요소.

<video> 태그.
<audio> 태그의 사용 방법과 거의 동일. ( src, autoplay, controls, loop, muted, preload )
width : 폭.
height : 높이.
poster : 지정한 비디오 데이터가 설정되어 있지 않거나 로딩되는 동안에 보여줄 이미지를 지정.

웹에서 주로 사용되는 비디오 형식.
.mp4 : MPEG-4에서 규정된 비디오 파일 형식.
.webm : 구글이 개발한 고화질 영상 압축 형식.
.ogv : 멀티미디어 표현을 위한 공개 소스 기반의 파일 형식.

브라우저별 지원 비디오 파일 형식.
각 브라우저/버전마다 지원되는 파일 형식이 다름.

<source> 태그.
오디오와 똑같이 사용할 수 있다.

비디오 재생 구간 지정.
미디어 컨텐츠를 재생할 때 특정 구간을 지정하여 재생.
<audio> , <video> 또는 <source> 태그의 src 속성 이용.
src = " 파일명#t = [ 시작시간 ] [ ,종료시간 ] "
시간은 '시 : 분 : 초' 로 표현

<track> 태그.
여러 언어나 비디오 설명을 제공하거나 또는 자막이나 캡션 같이 시간이 지정된 텍스트 파일을 지정.
<audio> , <video> 태그의 하위 태그로 사용. ( 단독으로 사용할 수 없다 )
src : 텍스트 트랙 파일의 url 지정. ( .vtt )
kind : 텍스트 트랙의 종류 지정. ( subtitles - 기본값 )
label : 사용자가 읽을 수 있는 트랙의 제목 지정.
srclang : 텍스트 트랙 데이터의 언어 지정. ( ko , en , ... 등 )
default : 트랙을 지정하지 않았을 때 사용할 기본 트랙을 나타낸다.

kind 속성.
subtitles : 비디오의 자막을 정의.
captions : 대화나 사운드 효과를 글로 옮기거나 번역한 것.
descriptions : 비디오 내용에 대한 텍스트 형식의 설명을 정의.
chapters : 장의 제목을 정의.
metadata : 스크립트에서 사용하기 위한 내용을 정의.

트랙 파일의 형식.
WebVTT Web Video Text Tracks : UTF-8 텍스트 형식 , 시작시간 종료시간과 자막을 묶어서 표현.

다중 트랙의 사용.
다중 트랙 설정시 default 속성을 사용해서 하나의 트랙을 기본으로 지정.
