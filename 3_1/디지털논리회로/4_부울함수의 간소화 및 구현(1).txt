
 부울함수의 간소화 및 구현(1).


 # 카르노 도표를 이용한 간소화.

부울함수의 간소화 방법.
대수적인 방법.
	● 주어진 부울함수에 부울대수의 정리를 대수적으로 적용하여 간소화.
	● 도표방법과 테이블 방법의 이론적 방법.
도표 방법.
	● 카르노 도표를 사용하는 방법.
	● 카르노 도표를 사용하여 부울함수의 각 항들을 곱이나 합 형태로 간소화.
	● 여섯 개 이하의 변수를, 가진 부울함수에 사용. ( 2~4변수에 사용 )
테이블 방법.
	● 퀸-맥클러스키 방법.
	● 테이블을 사용하여 간소화 알고리즘을 구현한 방법.
	● 많은 변수를 가진 부울함수에 적함.
	● 이번 과정에서는 배우지 않음.

카르노도표 방법의 개요.
개요.
	● 카르노 도표는 여러 개의 사각형으로 된 다이어그램.
	● 사각형 각각 하나의 최소항 또는 최대항을 의미.
	● 최소항 또는 최대항들이 차지하는 도표내의 면적을 이용하여 간소화.
	● 카르노 도표는 부울함수의 입력변수의 수에 따라 기본 도표의 형태가 결정.
	● 입력변수의 수가 n인 경우, n 변수 카르노 도표라 하고, 2**n개의 사각형으로 구성.
카르노 도표 사용.
	● 정규형 부울함수 => 표준형 부울함수로 간소화.
	● 최소항의 합 형태가 곱의 합 형태나, 최대항의 곱 형태가 합의 곱 형태로 간소화 됨.

최대항의 합형을 곱의 합형으로 간소화하는 순서.
1. 입력변수의 수 n에 따라 n변수 카르노 도표 작성( 도표는 2**n개의 정사각형 )
2. 최소항의 인덱스에 대응되는 사각형을 1로 표시.
3. 1로 표시된 사각형들 중 서로 인접한 사각형끼리 묶음.
4. 각 묶음이 입력변수 각각에 대해 도표상의 어떤 위치에 있는지 파악.
5. 4에서 구한 각 묶음에 대한 곱항들을 논리합으로 연결시키면 간소화된 표준형이 구해진다.
( 이미지 )

최대항의 곱형을 합의 곱형으로 간소화하는 순서.
( 이미지 )

인접 사각형의 정의.
카르노 도표에서 각 정사각형은 하나의 최소항 또는 최대항을 의미.
인접 사각형.
	● 두 정사각형에 대응되는 각 최소항 또는 최대항의 구성변수 중 다른 모든 변수는 동일하되
	   "오직 하나의 변수만 서로 보수관계에 있을 때 두 정사각형은 서로 인접한다" 라고 정의.
	● 대각선으로는 인접한다고 표현하지 않는다.

인접 사각형끼리 묶는 방법.
1. 한 묶음 내의 정사각형 수는 2**n 개가 되도록 묶는다.
2. 한 묶음은 크게, 전체 묶음의 수는 적게 묶는다.



 # 2, 3, 4 변수 카르노 도표.

두 개의 변수를 가지는 부울함수.
2변수 카르노 도표는 4개의 정사각형으로 구성된다.
각각의 정사각형은 하나의 최소항 또는 최대항에 대응.

세 개의 변수를 가지는 부울함수.
3변수 카르노 도표는 8개의 정사각형으로 구성된다.
각각의 정사각형은 하나의 최소항에 대응.
00, 01, 11, 10 으로 순서가 바뀐다.
왼쪽 끝과 오른쪽 끝의 값도 서로 인접해 있다.
