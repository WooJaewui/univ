
 레지스터와 카운터(1)



 # 동기식 카운터.

비동기식 카운터.
● 구조가 간단하고 동작이 단순.
● 동시에 트리거되지 않으므로 속도에 제약.

동기식 카운터.
● 모든 F/F에 동시에 클럭이 가해진다.
● 한꺼번에 동작되므로 동작속도의 향상.
	○ 2진 카운터.
	○ 모듈로-N 카운터.

2진 카운터.
2진 순서를 따르는 카운터.
	● 4비트 동기식 2진 카운터( 상향계수 )

모듈로-N 카운터.
N개의 계수 순서( 0, 1, 2, ... , N-1 )을 반복하여 계수하는 카운터.



 # 시프트 카운터.

시프트 카운터.
시프트 레지스터의 동작을 응용한 카운터.
종류.
	● 링 카운터.
	● 존슨 카운터.

링 카운터.
출력 비트 중 한 비트만 1이 되고 입력펄스에 의해 한쪽 방향으로 1의 위치가 순환.

링 카운터 계수 동작.
N개의 F/F으로 구성된 링 카운터는 N가지의 출력상태를 나타낸다.

존슨 카운터.
링 카운터와 유사.
다른 점은 첫 번째 F/F의 J, K 입력에 피드백되는 네 번째 F/F의 Q와 -Q가 반대로 연결.

존슨 카운터의 계수 동작.
존슨 카운터의 계수 동작은 링 카운터의 2배.
N개의 F/F을 사용하면 2N개의 출력상태가 나타난다.



 # 카운터의 설계.

카운터의 설계.
동기식 카운터 설계는 순서논리회로 설계와 동일.
종류.
	● 2진 카운터.
	● BCD 카운터.
	● 임의의 순서를 가진 카운터.

2진 카운터 설계. ( 0 ~ ( 2**N )-1 까지 )
1. 주어진 상태도에서 상태표 작성.
2. 상태표에서 카르노 도표를 이용하여 입력방정식 유도.
3. 입력방정식을 이용하여 논리도 작성.

BCD 카운터의 설계. ( 0 ~ 9까지 )
1. 주어진 상태도에서 상태표 작성.
2. 상태표에서 카르노 도표를 이용하여 입력방정식 유도.
3. 입력방정식을 이용하여 논리도 작성.
	● 출력 값과 논리회로를 추가해서 두 자리 세 자리 데이터도 카운트할 수 있다.

임의의 순서를 가진 카운터의 설계. ( 숫자가 정해지지 않음 )
1. 주어진 상태도에서 상태표 작성.
2. 상태표에서 카르노 도표를 이용하여 입력방정식 유도.
3. 입력방정식을 이용하여 논리도 작성.
	● 상태 값이 순서대로 되어 있지 않은 경우를 의미한다. ( 0, 1, 3, 5, 7, ... )
