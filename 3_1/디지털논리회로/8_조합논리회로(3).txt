
 조합논리회로(3)



 # MSI를 이용한 조합논리회로 - 디코더.

디코더 decoder.
부호화된 입력을 받아서 부호화되지 않은 출력을 내보내는 복호화기.
기계어에서 사람이 이해할 수 있는 언어로 바꾸는 과정.
n비트의 2진 코드를 최대 2**n 개의 서로 다른 정보로 바꾸어 주는 조합논리회로.

디코더의 확장.
작은 디코더를 여러 개 결합하여 필욯나 크기의 디코더 구성.
	● 예) 2개의 2 x 4 디코더를 이용하여 3 x 8 디코더를 구성할 수 있다.

디코더를 이용한 부울함수의 구현.
디코더는 n개의 입력변수에 대한 2**n 개의 최소항을 만든다.
따라서 부울함수는 최소항의 합으로 표현되므로, 디코더를 이용하면 부울함수를 구현할 수 있다.
n개의 입력과 m개의 출력을 가진 조합논리회로는 n x ( 2**n ) 디코더와 m 개의 OR 게이트로 만들 수 있다.

BCD-3초과 코드변환기를 디코더를 이용하여 구현.
1. 디코더와 OR 게이트로 구현할 때는 최소항의 합 형태로 표현.
2. 최소항의 합 형태는 진리표나 도표를 이용하여 구함.

멀티플렉서 multiplexer.
여러 개의 입력선 중에서 하나를 선택하여 단일의 출력을 내보내는 조합논리회로.
특정 입력선을 선택하기 위해서 선택변수를 사용.
2**n 개의 입력선 중에서 특정 입력선을 선택하기 위해서는 n개의 선택변수가 있어야 한다.
데이터 선택기 data selector라고도 하며, 약어로 MUX로 표현.

멀티플렉서에서 디코더 기능.
선택변수에 디코더의 최소항을 출력하는 특성을 멀티플렉서에 사용.
8개의 입력과 4개의 출력은 4개의 2x1 멀티플렉서 사용.

멀티플렉서를 이용한 부울함수 구현.
멀티플렉서는 OR 게이트를 가진 디코더와 같은 기능 수행.
n개의 선택입력과 2**n 개의 데이터 입력을 가진 멀티플렉서를 이용하면 n+1개의 변수를 가진 부울함수 구현 가능.

디멀티플렉서 demultiplexer.
멀티플렉서와 반대되는 연산을 수행하는 조합논리회로.
한 개의 입력선으로부터 정보를 받아 2**n 개의 출력선 중의 하나로 내보낸다.
이때 특정 출련선의 제어는 n개의 선택입력의 조합으로 제어된다.
디멀티플렉서와 디코더는 유사하다.



 # 정리.

MSI를 이용한 조합논리회로.
1. 인코더 : 부호화되지 않은 입력을 받아서 부호화된 출력을 내보내는 부호화기.
2. 디코더 : 부호화된 입력을 받아서 부호화되지 않은 출력을 내보내는 복호화기.
3. 멀티플렉서 : 여러 개의 입력을 받아서 단일의 출력을 내보내는 데이터 선택기.
4. 디멀티플렉서 : 하나의 입력을 받아서 여러 개의 출력을 내보내는 데이터 분배기.
